# simple-api-crud

## list of basic routes for foods:

| routes        | HTTP   | description     |
|---------------|--------|-----------------|
| /food     | GET    | get all data    |
| /food/:id | GET    | get one data    |
| /food     | POST   | create data     |
| /food/:id | DELETE | delete one data |
| /food/:id | PUT    | update one data |
